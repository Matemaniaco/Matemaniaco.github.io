Lenguaje matemático

Existen algunos comandos o expresiones que nos ayudan a escribir en lenguaje matemático. Aquí te presentaré algunas de ellas y te las explicaré brevemente, pero puedes encontrar fácilmente muchos manuales que te ayudarán a completar la información.

Comando Descripción Ejemplo Resultado
\sqrt{} sqrt es <>, raíz cuadrada, y entre las llaves se escribe el número \sqrt{7} √7
\sqrt[]{} cuando al comando sqrt se le añade otra llave, la primera llave es el índice y la segunda el número al que se le hace la raíz \sqrt[4]{5} ⁴√5
^{} ^{} sirve para elevar un número a otro 8^{12} 8¹²
\frac{}{} Sirve para escribir fracciones, en la primera llave se escribe el numerador y en la segunda el denominador \frac{2}{5} 2/5
\dfrac{}{} Sirve para escribir fracciones, en la primera llave se escribe el numerador y en la segunda el denominador \dfrac{8}{7} 8/7
\int_{}^{} int sirve para escribir integrales, _{} es el extremo inferior y ^{} es el extremo superior \int_{1}^{5} \(\int_{1}^{5} \)
\sum sum es el comando de la sumatoria \sum x_{i} Σxᵢ
\prod prod es el comando del producto \prod x_{i} ∏xᵢ
\pi número pi \pi π
\alpha letra griega alfa \alpha α
\beta letra griega beta \beta β
\gamma letra griega gamma \gamma γ
\sigma letra griega sigma, se usa para la desviación típica \sigma σ
